# Attendance-System
Expressフレームワークを使った出席登録システム（現在開発中）

## 概要
Expressを用いた出席登録・閲覧システムです。  
現在は開発中ですが、Attendance-botと連携し、Slackに対してメッセージを飛ばすことで簡単に出欠登録ができることを目指します。  
予定作成者は予定の作成・削除・編集が可能です。

データベース・・・SQL(PostgreSQL)
